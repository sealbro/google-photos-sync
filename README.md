# google-photos-sync

Sync google photos service
Easy photos/albums move between two accounts

## dev

- Set ENVs
  - `GOOGLE_CLIENT_ID`
  - `GOOGLE_SECRET_ID`
  - `PG_CONNECTION`
- Before launch
  - `swag init`
- [swagger + echo](https://github.com/swaggo/echo-swagger#start-using-it)
